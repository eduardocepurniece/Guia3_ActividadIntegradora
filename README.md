## Descripción del Proyecto
Este proyecto corresponde a la **Actividad Integradora 3** de la asignatura de Base de Datos, en la cual se desarrolla un **Sistema de Inteligencia Comercial** para una empresa de distribución.

El propósito es aplicar los conocimientos sobre:
- Consultas multitablas  
- Consultas avanzadas  
- Vistas  
- Procedimientos almacenados  

Todo esto orientado a generar información útil para la **toma de decisiones empresariales**.

---

## Objetivo General
Aplicar de forma integrada los conceptos de SQL para analizar datos empresariales y construir soluciones orientadas a la optimización del negocio.
